## 问题


### 玩家在什么情况下不能动？
- 玩家前面是墙
- 玩家前面有多个箱子
- 玩家前面是箱子+墙

### 如何推动箱子？
判断玩家前面有没有箱子，如果有的话，再判断箱子是否可以推动，如果可以推动的话，就同时移动箱子和玩家


### 箱子在什么情况下不可以推动？
- 箱子前面是墙
- 箱子前面是箱子


### 游戏胜利条件是什么？
胜利条件是每个目标点上都有箱子

### 目标点上有箱子/玩家时怎么切换展示的图片？
通过`isTargetPos`，传入坐标，判断该坐标是否为目标点，如果是的话，就切换图片

### 地图编辑器？
编辑器需要编辑地图、设置玩家初始位置、设置箱子初始位置、设置目标点
